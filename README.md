# 圖片壓縮 · Image Compressor

一個純前端的圖片壓縮小工具。所有運算都在瀏覽器本地完成，**圖片不會上傳到任何伺服器**，速度快又保護隱私。

## 功能

- 上傳 **PNG / JPG / WebP**，可拖曳或點擊，支援一次多張
- 兩種壓縮模式：
  - **手動調整**：自由設定壓縮品質與尺寸縮放
  - **目標大小**：輸入目標 KB（例如 200），自動找出不超過上限、畫質最佳的設定
- 可選輸出格式：JPEG（最小）／ WebP（高壓縮率）／ PNG（無損、保留透明）
- 即時顯示壓縮前後的預覽、檔案大小與節省比例
- **壓縮前後滑動對比**：拖曳分隔線直接比較畫質差異
- 單張下載，或一鍵 **打包成 ZIP** 下載全部

## 使用方式

不需安裝、不需後端，二選一：

### 方式 A：直接開啟（最簡單）
解壓資料夾後，**雙擊 `index.html`** 即可在瀏覽器使用。

### 方式 B：本機啟動小型伺服器（選用）
若想以網址方式開啟，可在資料夾內執行：

```bash
python -m http.server 8123
```

然後瀏覽器開啟 <http://localhost:8123>。

> 注意：`localhost` 網址只有你自己的電腦能開啟。要分享給他人請見下方「分享給夥伴」。

## 分享給夥伴

整個程式只有以下檔案，沒有任何安裝依賴：

```
圖片壓縮網站/
├─ index.html
├─ styles.css
├─ app.js
└─ README.md
```

- **直接傳檔**：把資料夾壓成 zip 給對方，對方解壓後雙擊 `index.html` 就能用。
- **給公開網址**：把檔案上傳到任一免費靜態託管（GitHub Pages、Netlify、Vercel、Cloudflare Pages）即可，免費且自帶 HTTPS。

## 存取保護（通行碼）

網站開啟時會要求輸入**通行碼**，正確後才能使用，並在當次瀏覽器分頁記住（不需每次重整都輸入）。通行碼請私下告知夥伴，不要寫在公開文件或截圖中。

> 說明：這是**前端通行碼**，以 SHA-256 雜湊存放（明碼不寫在程式裡），用途是擋住一般陌生人與路過誤入者；它無法達到高安全性（懂技術者仍可繞過）。若需更換通行碼，請以新碼的 SHA-256 值替換 `app.js` 中的 `PASSCODE_HASH`。

## 使用小提醒

- **照片**請用 **JPEG** 或 **WebP**；需要透明背景或截圖/線稿再用 **PNG**。
- 把已是 JPG 的照片轉成 PNG 通常**會變大且不會更清晰**——畫質上限取決於原圖，無損格式只是完整保存現有像素，無法找回已失去的細節。
- 想直接控制檔案大小時，改用「目標大小」模式最方便。

## 瀏覽器需求

使用較新的瀏覽器即可（Chrome / Edge / Safari / Firefox 近年版本）。WebP 輸出需瀏覽器支援，現代瀏覽器皆已支援。

## 技術說明

- 純 HTML / CSS / JavaScript，**無第三方依賴、無建置流程**。
- 以 Canvas API 進行解碼、縮放與重新編碼。
- ZIP 打包為自行實作的 store（不壓縮）寫入器（圖片本身已壓縮，再壓收益有限）。
