## 一、改完網站要上線（最常用）

在專案資料夾的終端機，依序貼上：

```bash
git add .
git commit -m "說明這次改了什麼"
git push
```

- 推送後 **GitHub Pages 自動重新部署**，約 **1～2 分鐘**生效。
- 看不到更新時：用**無痕視窗**或按 **Ctrl + F5** 強制重新整理（多半是瀏覽器快取）。

## 二、更換通行碼

1. 算出新通行碼的 SHA-256（PowerShell，把 `新通行碼` 換成你要的字）：

```powershell
$s = "新通行碼"
$b = [System.Text.Encoding]::UTF8.GetBytes($s)
$h = [System.Security.Cryptography.SHA256]::Create().ComputeHash($b)
(($h | ForEach-Object { $_.ToString("x2") }) -join "")
```

2. 複製跑出來的那串 64 字元雜湊。
3. 打開 `app.js`，找到 `PASSCODE_HASH`，把引號裡的舊雜湊換成新的。
4. 執行上面「一、改完網站要上線」的三步驟推送。

> 也可以直接把新通行碼告訴 AI 助手，請它幫你換好。
